# Calculate time series metrics

Generic function for `calc_ts_metrics.cube` and
`calc_ts_metrics.spatRaster`

## Usage

``` r
calc_ts_metrics(x, ...)
```

## Arguments

- x:

  data cube or spatRaster

- ...:

  additional arguments passed to `calc_ts_metrics.cube` or
  `calc_ts_metrics.rast`

## Details

Currently, there are some differences in the inputs/outputs of the
functions for data cube and spatRaster, this has to be harmonized.
