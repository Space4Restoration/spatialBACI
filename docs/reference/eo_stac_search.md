# Search STAC EO data

Easier searching for EO data in STAC collections

## Usage

``` r
eo_stac_search(endpoint, ...)
```

## Arguments

- endpoint:

  see \[rstac::search()\]

- ...:

  see \[eo_stac_search.PlanetaryComputer()\]

## Value

STACItemCollection

## Details

For now only implemented for Planetary Computer
