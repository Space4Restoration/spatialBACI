# OpenStreetMap distance

Calculated shortest geographic distance to selected OpenStreetMap
features

## Usage

``` r
osm_distance(x, what, ...)
```

## Arguments

- x:

  SpatRaster or SpatVector for which distances are calculated

- what:

  Either "roads" or "places"

- ...:

  Additional arguments to osm_distance_roads or osm_distance places

## Details

See osm_distance_roads, osm_distance places
