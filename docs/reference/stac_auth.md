# STAC authentication

Authenticate STAC items for specified STAC endpoint

## Usage

``` r
stac_auth(x, endpoint, authOpt = list())
```

## Arguments

- x:

  a STAC Feature or FeatureCollection

- endpoint:

  a STAC endpoint

- authOpt:

  additional arguments for authentication options specific to STAC
  endpoint (e.g., list(key="abc"))

## Value

description

## Details

Implemented endpoints: \* Microsoft Planetary Computer
